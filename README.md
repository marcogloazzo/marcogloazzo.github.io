# Página personal / CV académico

Este es un sitio web simple para publicar con GitHub Pages.

## Archivos principales

- `index.html`: contenido de la página.
- `style.css`: diseño visual.
- `cv.pdf`: tu CV descargable. Todavía no está incluido; tenés que agregarlo vos con ese nombre.

## Cómo publicarlo en GitHub Pages

1. Crear una cuenta en GitHub.
2. Crear un repositorio llamado exactamente:

   `TU-USUARIO.github.io`

3. Subir estos archivos al repositorio:
   - `index.html`
   - `style.css`
   - `cv.pdf`, si ya tenés tu CV listo.

4. Ir a:

   `Settings` → `Pages`

5. En `Build and deployment`, elegir:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`

6. Guardar.

Luego la página debería estar disponible en:

`https://TU-USUARIO.github.io`

## Cosas que conviene editar

En `index.html`, buscá y reemplazá:

- `your.email@example.com`
- `your-github-username`
- `your-linkedin-user`

También podés modificar la sección de proyectos, habilidades y descripción personal.
